The original datasets needs to be downloaded and placed in the `data/original/` directory. E.g.:`data/original/[DATASET]`

After downloading, use the scripts in `scripts/` to process the data and generate the files specific to the study. After processing, the files will be saved in `data/processed/[DATASET]/`.

Sources:
- Kermany dataset: [Link](https://data.mendeley.com/datasets/rscbjbr9sj/3)
- RSNA Pneumonia Detection Challenge: [Link](https://www.kaggle.com/competitions/rsna-pneumonia-detection-challenge/data)
